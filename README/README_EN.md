## 🤝 Contribute!

**This project is fully open-source.** We sincerely welcome all contributions!

Whether you want to add new sites, improve the UI, develop features, or share ideas — all are welcome!

### How to Contribute

- **Add Sites**: Found a great, stable site? Submit an Issue or Pull Request!
- **UI Improvements**: Think the design, colors, or layout could be better? Go ahead!
- **Feature Development**: Ideas like enhanced search, favorites, tags, theme switching, i18n — all welcome!
- **Bug Reports**: Found any issue? Let us know!

**Please open an Issue first before submitting a PR** to avoid duplicate work.
